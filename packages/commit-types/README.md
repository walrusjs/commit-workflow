## 类型

- feat: 引入新功能
- fix: 修复 bug
- chore: 使用配置或其他内容
- test: 添加单元测试或 e2e 测试
- wip: 进行中的工作
- docs: 添加或修改文档
- refactor: 重构代码（不引入功能或修复）
- revert: 还原更改或提交
- style: 更新 UI 和样式文件
- perf: 提高性能
- ci: 关于 CI 构建系统的工作
